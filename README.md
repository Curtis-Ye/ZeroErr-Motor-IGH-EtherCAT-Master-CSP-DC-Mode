# ZeroErr-Motor-IGH-EtherCAT-Master-CSP-DC-Mode
本项目目的是研究零差云控关节模组在CSP模式下开启DC同步时钟。  
官方提供了一套CSV模式的代码，打算在该示例代码的基础上进行研究与开发。  
首先我将会分析这份代码的结构，分块进行全面解析。二次开发是建立在理解原理的基础上。官方代码以及CMake文件我都会上传到src文件夹的子目录下。  
除了对代码进行解析之外，同时我也会拓展一些EtherCAT相关的知识。  
本项目使用的软硬件如下：零差云控关节模组x1、零差云控800W电源x1、Ubuntu系统主机x1、RJ45网线关节模组转接线x1 。软件采用IGH EtherCAT Master。
**本项目会一直更新**
