# 相关功能函数解析
本节介绍功能函数的功能。
首先是`ec_master_t *master;`这行代码可以理解为创建了一个主站对象。后续很多地方都要用到这个对象。  
```
void ODwrite(ec_master_t *master, uint16_t slavePos, uint16_t index, uint8_t subIndex, uint8_t objectValue)
{
	uint8_t retVal = ecrt_master_sdo_download(master, slavePos, index, subIndex, &objectValue, sizeof(objectValue), NULL);
	if (retVal)
		printf("OD write unsuccessful\n");
}
```
ODwrite函数用于配置PDO映射，其实在EtherCAT中PDO映射的配置是通过SDO来完成的，所以ODwrite函数里面封装的ecrt_master_sdo_download函数通过这个函数的名字就可以看出是通过SDO来进行配置的。  
里面各参数的含义：首先master是一个ec_master_t类型的指针变量，传入主站对象；其次slavePos是从站位置，这个暂且不展开；下面两个index和subIndex分别是索引和子索引。具体如何理解呢？首先EtherCAT里有一个对象字典，里面保存了所有参数，每个参数都有自己的地址。这个地址就是index + subIndex。其中index是一个大类，subIndex是其中的细分。本质上是告诉主站：我要把哪个对象映射进PDO。最后一个参数：objectValue是要写入这个PDO对象的值。  
```
void initDrive(ec_master_t *master, uint16_t slavePos)
{
	ODwrite(master, slavePos, 0x6060, 0x00, 0x09); // 0x09 for CSV mode
	/* Reset alarm */
	ODwrite(master, slavePos, 0x6040, 0x00, 0x80);
}
```
这个函数的功能通过函数名也可以看出来，是来初始化关节模组的。参数就不再介绍，与上一个函数相同。重点看内部封装的函数，封装的函数都是ODwrite函数，第一个是给索引为0x6060的参数赋予0x09的值，查询对象字典可知，这行代码的目的是设置模组的工作模式是CSV模式。第二行同理，是将关节模组的控制字设为0x80，用于复位。  
```
inline void timespec_add(struct timespec *result, struct timespec *time1, struct timespec *time2)
{

	if ((time1->tv_nsec + time2->tv_nsec) >= NSEC_PER_SEC)
	{
		result->tv_sec = time1->tv_sec + time2->tv_sec + 1;
		result->tv_nsec = time1->tv_nsec + time2->tv_nsec - NSEC_PER_SEC;
	}
	else
	{
		result->tv_sec = time1->tv_sec + time2->tv_sec;
		result->tv_nsec = time1->tv_nsec + time2->tv_nsec;
	}
}
```
这个函数用于计算两个时间和，timespec是`<time.h>`文件中定义的结构体。结构体的内容如下：  
```
struct timespec {
    time_t tv_sec;   // 秒
    long   tv_nsec;  // 纳秒
};
```
下一个函数为：  
```
inline void timespec_sub(struct timespec *result, struct timespec *time1, struct timespec *time2)
{

	if ((time1->tv_nsec - time2->tv_nsec) < 0)
	{
		result->tv_sec = time1->tv_sec - time2->tv_sec - 1;
		result->tv_nsec = NSEC_PER_SEC - (time1->tv_nsec - time2->tv_nsec);
	}
	else
	{
		result->tv_sec = time1->tv_sec - time2->tv_sec;
		result->tv_nsec = time1->tv_nsec - time2->tv_nsec;
	}
}
```

	kill(pid, SIGKILL);

这是用于进程管理的函数。主要功能是用于释放当前进程。  
```
void stack_prefault(void)
{
	unsigned char dummy[MAX_SAFE_STACK];
	memset(dummy, 0, MAX_SAFE_STACK);
}
```
memset是一个初始化函数，作用是将某一块内存中的全部设置为指定的值。stack_prefault函数用于初始化一片内存空间。  
```
uint16_t getDriveState(uint16_t statusWord)
{
	return statusWord & 0x6f; // Mask to get state bits
}
```
这个函数顾名思义就是用来获取关节模组的状态。
