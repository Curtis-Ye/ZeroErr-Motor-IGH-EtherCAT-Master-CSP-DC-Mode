这个代码块用于配置域（domain）。域就是创建一个PDO数据缓冲区。PDO决定“传什么数据”，Mapping决定“PDO里放哪些对象”，而Domain则是IGH EtherCAT主站中用于存放这些PDO数据的一块共享内存缓冲区。

```
#ifdef CONFIG_DC
	struct timespec t;
	clock_gettime(CLOCK_MONOTONIC, &t);
	ecrt_master_application_time(master, EC_NEWTIMEVAL2NANO(t));
	/* Do not enable Sync1 */
	ecrt_slave_config_dc(drive0, 0x0300, PERIOD_NS, 0, 0, 0); // 配置从站的分布式时钟参数
#endif
```
这个代码块首先声明了一个timespec的结构体变量t，然后使用`clock_gettime`函数用于获取时间。其中，"clock_gettime"是基于Linux C语言的时间函数,可以用于计算精度和纳秒。`CLOCK_MONOTONIC`是一个宏定义，表示从系统启动这一刻起开始计时,不受系统时间被用户改变的影响。本函数用于获取当前系统时间。`ecrt_master_application_time`函数负责把“主站认为当前应该是多少时间”告诉 EtherCAT 网络。`ecrt_slave_config_dc`表示启用DC、启用SYNC0、SYNC0周期：PERIOD_NS、偏移：0ns。  
```
#ifdef SYNC_REF_TO_MASTER
	/* Initialize master application time. */
	struct timespec masterInitTime;
	clock_gettime(CLOCK_MONOTONIC, &masterInitTime);
	ecrt_master_application_time(master, TIMESPEC2NS(masterInitTime));
#endif
```
与上面代码块类似。上一次用于给 `ecrt_slave_config_dc()` 提供时间基准。这一次用于`ecrt_master_sync_reference_clock()` 提供初始时间。
